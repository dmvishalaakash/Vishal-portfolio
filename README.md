# Vishal | Generative AI Specialist Portfolio

A high-end, futuristic portfolio showcasing expertise in Generative AI, AI Content Creation, and Technical Validation.

## 📁 Assets
The professional resume is located in `/public/assets/resume.pdf`.
(Last Sync Check: 2026-03-08 14:21 - Renamed to resume.pdf)

## 🚀 Features

- **Futuristic UI**: Cyberpunk-inspired design with fluid animations and neural backgrounds.
- **Theme System**: Multiple cosmic themes (Interstellar, Cyberpunk, Nebula, Deep Space).
- **Responsive Design**: Fully optimized for all devices.
- **Resume Integration**: Direct view and download options for the professional resume.
- **Tech Stack**: React 19, Vite, Tailwind CSS, Motion (Framer Motion), and Lucide React.

## 🛠️ Installation & Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/vishal-ai-portfolio.git
   cd vishal-ai-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

## 📦 Deployment

### Deploy to Vercel

This project is optimized for Vercel.

1. Push your code to GitHub.
2. Connect your GitHub repository to [Vercel](https://vercel.com).
3. Vercel will automatically detect the Vite settings and deploy your site.

**Note**: The `vercel.json` file is already configured to handle routing and static assets correctly.

## 📂 Project Structure

- `public/assets/`: Contains static assets like the resume PDF.
- `src/components/`: Reusable UI components.
- `src/App.tsx`: Main application logic and layout.
- `src/index.css`: Global styles and Tailwind configuration.
